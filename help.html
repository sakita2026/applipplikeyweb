ワンタイムパスワードによるログインLogging in with OTP

ワンタイムパスワード初期設定が完了すると、Windows ログイン画面でワンタイムパスワードを求められます。スマートフォンのワンタイムパスワードアプリに表示された 6桁のワンタイムパスワード を入力することにより、安全にログインすることができます。 After OTP setup is complete, the Windows login screen will ask for a one-time password. Enter the 6-digit OTP displayed in your authenticator app to log in securely.

ログインユーザー名は入力せず、ワンタイムパスワード入力のみでログインできます。 ただし、前回ログインしたユーザー名を記憶しているため、別のユーザーでログインする場合はユーザーの切り替え操作が必要です。 また、複数ユーザーで applippli-key を設定している場合は、ユーザー名欄を空白のままそれぞれのワンタイムパスワードを入力すればそのユーザーでログインできます。 You can log in by entering only the OTP — no username is required. The system remembers the last logged-in username, so switching to a different user requires a user switch operation. If multiple users have applippli-key configured, each user can log in by leaving the username field blank and entering their own OTP.

緊急ログオンの使用方法Using Emergency Logon

ワンタイムパスワードアプリを設定したスマートフォンを忘れた・故障した場合など、ワンタイムパスワードの確認ができない場合に操作します。 Use this procedure when you cannot access your OTP — for example, if you have forgotten or lost your smartphone.

  • 1 PC のログイン画面上に表示されている ［緊急ログオン］ をクリックします。 Click [Emergency Logon] on the Windows login screen.
  • 2 PC のユーザー名を入力し、［次へ］ をクリックします。 Enter your PC username and click [Next].
  • 3 セットアップ時に入力した緊急ログオン設定での回答内容を入力し、［次へ］ をクリックします。
    ※ 半角英数字のみで入力してください。     Enter the answer you provided during Emergency Logon setup, then click [Next].
    ※ Use half-width alphanumeric characters only.
  • 4 処理完了のメッセージが表示されたら ［OK］ で閉じます。 When the completion message appears, click [OK] to close it.
  • 5 Windows パスワード入力画面に切り替わったら、設定している Windows パスワードを入力し、ログインします。 When the Windows password screen appears, enter your Windows password to log in.

ワンタッチログインの設定One-Touch Login Setup

ワンタッチログインは applippli-key Ver 2 から追加された機能です。スマートフォンアプリと連携することで、スマートフォンのワンタッチ操作だけで Windows にログオンできます。 One-Touch Login is a new feature in applippli-key Ver 2. By pairing with the smartphone app, you can log on to Windows with a single tap on your phone.

事前準備Prerequisites

  • 1 スマートフォンに applippli-key アプリ（iOS / Android）をインストールします。 Install the applippli-key app (iOS / Android) on your smartphone.
  • 2 applippli-key Ver 2 がインストール済みであることを確認します。 Confirm that applippli-key Ver 2 is installed on your PC.

ペアリング手順Pairing Steps

  • 1 applippli-key の設定画面を開き、「ワンタッチログイン設定」 をクリックします。 Open the applippli-key settings screen and click "One-Touch Login Setup."
  • 2 Windows パスワードを入力します。 Enter your Windows password.
  • 3 ペアリング用の QR コードが表示されます。applippli authenticator アプリを起動して QR コードを読み取ります。 A pairing QR code will appear. Open the applippli authenticator app and scan the QR code.
  • 4 スマートフォンアプリ側でペアリングが完了すると、設定画面に 「スマートフォンが接続されました」 と表示されます。 Once pairing is complete in the smartphone app, the settings screen will show "Smartphone connected."

ワンタッチログインの使い方How to Use One-Touch Login

  • 1 Windows のログオン画面が表示されたら、スマートフォンにて applippli authenticator を起動します。 When the Windows logon screen appears, open the applippli authenticator app on your smartphone.
  • 2 リストよりログインする PC 名を 1 回だけタップします。 Tap the PC name you want to log in to once from the list.
  • 3 Windows に自動的にログオンします。 Windows will log on automatically.

ログビューアLog Viewer

ログビューアは applippli-key Ver 2 から追加された管理ツールです。MFA 認証ログの確認・分析に加え、脅威の自動検知・自動 IP ブロック・LINE 通知などのセキュリティ機能を提供します。 The Log Viewer is a management tool added in applippli-key Ver 2. In addition to viewing and analyzing MFA authentication logs, it provides security features such as automatic threat detection, automatic IP blocking, and LINE notifications.

起動方法How to Launch

  • 1 スタートメニューから 「applippli-key」 → 「applippli-key ログ」 をクリックします。 From the Start menu, click "applippli-key" → "applippli-key Log."

脅威自動検知Automatic Threat Detection

ログビューアはバックグラウンドで以下を常時監視します。検知した場合に LINE 通知（設定済みの場合）を送信します。 The Log Viewer continuously monitors the following in the background. A LINE notification is sent when a threat is detected (if configured).

監視対象Monitor Target 検知内容Detection Details
不審プロセス起動Suspicious Process mimikatz・psexec・rclone・BloodHound など既知の攻撃ツール 25 種の起動を検知Detects launch of 25 known attack tools such as mimikatz, psexec, rclone, and BloodHound
プロセス偽装（5種）Process Masquerading (5 types) ① ファイルタイプ偽装 — .txt/.jpg 等の拡張子で実行ファイルを偽装
② システムプロセス偽装 — svchost.exe 等が正規パス外または Microsoft 署名なし
③ 二重拡張子偽装 — 例: report.doc.exe
④ 不正署名 — AppData/Temp 等から署名検証に失敗するバイナリを実行
⑤ リネーム偽装 — 実行ファイルのバージョン情報（OriginalFilename）と実ファイル名が不一致 ① File type masquerade — executable disguised with .txt/.jpg extension
② System process masquerade — svchost.exe etc. outside legitimate path or without Microsoft signature
③ Double extension — e.g. report.doc.exe
④ Invalid signature — binary with failed signature verification run from AppData/Temp

LINE 通知の設定方法How to Set Up LINE Notifications

  1. ログビューアを起動し、「LINE通知設定」タブを開くLaunch Log Viewer and open the "LINE Notification Settings" tab
  2. LINE Notify または LINE 公式アカウントの連携トークンを入力Enter the LINE Notify or official account integration token
  3. 「テスト送信」ボタンで通知が届くことを確認Click "Send Test" to confirm the notification is received
  4. 通知したいイベント種別にチェックを入れて保存Check the event types you want to be notified about and save

自動 IP ブロックAutomatic IP Blocking

「自動防御設定」タブで有効にすると、ネットワーク不正アクセスを検知した送信元 IP を Windows ファイアウォールで自動ブロックします。 When enabled in the "Auto Block Settings" tab, the source IP of detected network intrusion attempts is automatically blocked by Windows Firewall.

設定項目Setting 説明Description
ブロック対象Block Target 検知した IP の /24 サブネット単位でブロック（IP 変更による回避に対応）Blocks the /24 subnet of the detected IP (to counter evasion via IP rotation)
ブロック期間Block Duration デフォルト 24 時間（1〜720 時間で設定可）または無期限。期限切れルールは 30 分おきに自動削除。Default 24 hours (configurable from 1–720 hours) or permanent. Expired rules are automatically removed every 30 minutes.
除外 IP レンジExcluded IP Ranges 社内ネットワーク（10.x / 172.16.x / 192.168.x 等）はデフォルトで除外。CIDR 形式で追加設定可。Internal networks (10.x / 172.16.x / 192.168.x etc.) are excluded by default. Additional ranges can be added in CIDR notation.
手動解除Manual Unblock 「自動防御設定」タブのリストから「今すぐ解除」ボタンで即時解除可。Click "Unblock Now" in the list on the "Auto Block Settings" tab to unblock immediately.
