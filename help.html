ワンタイムパスワードによるログインLogging in with OTP

ワンタイムパスワード初期設定が完了すると、Windows ログイン画面でワンタイムパスワードを求められます。スマートフォンのワンタイムパスワードアプリに表示された 6桁のワンタイムパスワード を入力することにより、安全にログインすることができます。 After OTP setup is complete, the Windows login screen will ask for a one-time password. Enter the 6-digit OTP displayed in your authenticator app to log in securely.

ログインユーザー名は入力せず、ワンタイムパスワード入力のみでログインできます。 ただし、前回ログインしたユーザー名を記憶しているため、別のユーザーでログインする場合はユーザーの切り替え操作が必要です。 また、複数ユーザーで applippli-key を設定している場合は、ユーザー名欄を空白のままそれぞれのワンタイムパスワードを入力すればそのユーザーでログインできます。 You can log in by entering only the OTP — no username is required. The system remembers the last logged-in username, so switching to a different user requires a user switch operation. If multiple users have applippli-key configured, each user can log in by leaving the username field blank and entering their own OTP.

緊急ログオンの使用方法Using Emergency Logon

ワンタイムパスワードアプリを設定したスマートフォンを忘れた・故障した場合など、ワンタイムパスワードの確認ができない場合に操作します。 Use this procedure when you cannot access your OTP — for example, if you have forgotten or lost your smartphone.

  • 1 PC のログイン画面上に表示されている ［緊急ログオン］ をクリックします。 Click [Emergency Logon] on the Windows login screen.
  • 2 PC のユーザー名を入力し、［次へ］ をクリックします。 Enter your PC username and click [Next].
  • 3 セットアップ時に入力した緊急ログオン設定での回答内容を入力し、［次へ］ をクリックします。
    ※ 半角英数字のみで入力してください。     Enter the answer you provided during Emergency Logon setup, then click [Next].
    ※ Use half-width alphanumeric characters only.
  • 4 処理完了のメッセージが表示されたら ［OK］ で閉じます。 When the completion message appears, click [OK] to close it.
  • 5 Windows パスワード入力画面に切り替わったら、設定している Windows パスワードを入力し、ログインします。 When the Windows password screen appears, enter your Windows password to log in.

同一 PC で複数ユーザーが存在する場合で他のユーザーへ切り替える場合、下記の手順にて他ユーザーでのログインができます。 If multiple users are registered on the same PC, follow the steps below to log in as a different user.

  • 1 ログイン画面左下にある ［その他のユーザー］ をクリックします。 Click [Other users] at the bottom left of the login screen.

ワンタッチログインの設定One-Touch Login Setup

ワンタッチログインは applippli-key Ver 2 から追加された機能です。スマートフォンアプリと連携することで、スマートフォンのワンタッチ操作だけで Windows にログオンできます。 One-Touch Login is a new feature in applippli-key Ver 2. By pairing with the smartphone app, you can log on to Windows with a single tap on your phone.

事前準備Prerequisites

  • 1 スマートフォンに applippli-key アプリ（iOS / Android）をインストールします。 Install the applippli-key app (iOS / Android) on your smartphone.
  • 2 applippli-key Ver 2 がインストール済みであることを確認します。 Confirm that applippli-key Ver 2 is installed on your PC.

ペアリング手順Pairing Steps

  • 1 applippli-key の設定画面を開き、「ワンタッチログイン設定」 をクリックします。 Open the applippli-key settings screen and click "One-Touch Login Setup."
  • 2 Windows パスワードを入力します。 Enter your Windows password.
  • 3 ペアリング用の QR コードが表示されます。applippli authenticator アプリを起動して QR コードを読み取ります。 A pairing QR code will appear. Open the applippli authenticator app and scan the QR code.
  • 4 スマートフォンアプリ側でペアリングが完了すると、設定画面に 「スマートフォンが接続されました」 と表示されます。 Once pairing is complete in the smartphone app, the settings screen will show "Smartphone connected."

ワンタッチログインの使い方How to Use One-Touch Login

  • 1 Windows のログオン画面が表示されたら、スマートフォンにて applippli authenticator を起動します。 When the Windows logon screen appears, open the applippli authenticator app on your smartphone.
  • 2 リストよりログインする PC 名を 1 回だけタップします。 Tap the PC name you want to log in to once from the list.
  • 3 Windows に自動的にログオンします。 Windows will log on automatically.

applippli-key Ver 2 のログビューアは、MFA 認証の監視に加え、ランサムウェア攻撃の前兆となる不審なプロセス・コマンドをリアルタイムで検知します。検知した内容は security_log.txt に記録され、ログビューアの「ログ」タブに表示されます。 In addition to MFA authentication monitoring, the applippli-key Ver 2 Log Viewer detects suspicious processes and commands that may indicate a ransomware attack in real time. Detected events are recorded in security_log.txt and displayed in the Log Viewer's "Logs" tab.

security_log.txt についてAbout security_log.txt

脅威を検知すると、ログビューアは security_log.txt に自動記録します。このファイルはログビューアの「ログ」タブに認証ログと混在して表示され、THREAT 行として暗赤色で強調されます。 When a threat is detected, Log Viewer automatically writes to security_log.txt. This file is displayed alongside authentication logs in the "Logs" tab, with THREAT entries highlighted in dark red.

// security_log.txt 記録フォーマット例
02/07/2026 ,19:42:11 ,プロセス検知 ,THREAT ,mimikatz.exe ,IP: ,PC:DESKTOP-ABC ,パスワード窃取ツール（mimikatz）を検知 , ,

検知時の対応手順Response Procedure on Detection

⚠ THREAT 検知時は即時対応が必要です⚠ Immediate action required on THREAT detection ランサムウェア攻撃はバックアップを破壊してから暗号化を行います。検知通知を受けた時点で速やかに対応することで被害を最小化できます。 Ransomware attacks destroy backups before encrypting files. Responding promptly upon receiving a detection notification minimizes damage.
  • 1 LINE 通知の 【対応】 フィールドに記載された対処内容を確認します。 Check the [Action] field in the LINE notification for recommended steps.
  • 2 対象 PC をネットワークから切断（LAN ケーブル抜去・Wi-Fi 無効化）します。 Disconnect the target PC from the network (unplug LAN cable / disable Wi-Fi).
  • 3 LogViewer の「ログ」タブで THREAT 行（暗赤色）を確認し、攻撃フェーズ・対象プロセス・実行時刻を記録します。 In the LogViewer "Logs" tab, locate THREAT entries (dark red) and note the attack phase, process name, and timestamp.
  • 4 システム管理者または社内セキュリティ担当者に報告し、フォレンジック調査や感染範囲の確認を行います。 Report to your system administrator or security team and proceed with forensic investigation and infection scope assessment.

前提条件Prerequisites

  • EventID 4688 のコマンドライン記録を有効化する必要があります。
    グループポリシー：コンピューターの構成 → Windows の設定 → セキュリティの設定 → 詳細な監査ポリシーの構成 → 詳細な追跡 →「プロセス作成の監査」を有効にし、「コマンドラインの記録」を有効化してください。     EventID 4688 command-line logging must be enabled.
    Group Policy: Computer Configuration → Windows Settings → Security Settings → Advanced Audit Policy → Detailed Tracking → Enable "Audit Process Creation" and enable "Include command line in process creation events."
  • Sysmon（LSASS アクセス検知用）を別途インストールすることで、より高精度な検知が可能になります。 Installing Sysmon separately (for LSASS access detection) enables higher-accuracy detection.
  • ログビューアが起動していない場合は検知・通知が行われません。Windows 起動時の自動起動設定を推奨します。 Detection and notifications do not occur if Log Viewer is not running. We recommend enabling auto-start at Windows startup.

LINE通知設定 LINE Notification Setup

LINE通知機能はアクティベーション済みライセンスが必要です。ライセンスが有効な場合、ログビューアの「LINE通知」タブに設定画面が表示されます。 The LINE notification feature requires an activated license. When your license is active, a "LINE Notification" tab appears in the Log Viewer.

概要Overview

applippli-key の認証イベント（ログイン成功・失敗・不正アクセス検知）をリアルタイムで LINE に通知する機能です。ログビューアが起動している間、認証ログを監視し自動で通知を送信します。 This feature sends real-time LINE notifications for applippli-key authentication events (login success, failure, and unauthorized access detection). Notifications are sent automatically while Log Viewer is running.

⚠ ご注意⚠ Note ログビューアが起動していない場合は通知は届きません。パソコン起動時に自動起動するよう設定することをお勧めします（後述）。 Notifications will not be sent if Log Viewer is not running. We recommend configuring it to launch automatically at Windows startup (see below).

① LINEボットを友達追加する① Add the LINE Bot as a Friend

  • 1 ログビューアを起動し、左メニューの 「LINE通知」 をクリックします。 Launch Log Viewer and click "LINE Notification" in the left menu.
  • 2 設定画面に表示されている QR コードを LINE アプリで読み取り、「applippli-key 通知サービス」 を友達追加します。 Scan the QR code shown in the settings screen using the LINE app to add "applippli-key 通知サービス" as a friend.
  • 3 友達追加すると、ボットから LINE User ID（例：U1234567890abcdef...）が自動返信されます。 After adding the bot, it will automatically reply with your LINE User ID (e.g., U1234567890abcdef...).
すでに友達追加済みの場合は、ボットにメッセージ（内容は何でも可）を送信すると LINE User ID が返信されます。 If you have already added the bot as a friend, simply send any message to the bot and it will reply with your LINE User ID.

② LINE User ID を設定して保存する② Enter and Save Your LINE User ID

  • 1 ボットから返信された LINE User ID を「LINE 通知」設定画面の入力欄に貼り付けます。 Paste the LINE User ID received from the bot into the input field in the "LINE Notification" settings screen.
  • 2 「保存」 ボタンをクリックして設定を保存します。 Click the "Save" button to save the settings.
  • 3 通知するイベントの種類（後述）にチェックを入れ、再度 「保存」 をクリックします。 Check the event types you want to be notified about (see below), then click "Save" again.
  • 4 「テスト送信」 ボタンをクリックして、LINE にテスト通知が届くことを確認します。 Click the "Test Send" button and confirm that a test notification arrives on LINE.

ログビューアの自動起動設定Auto-Start Log Viewer at Startup

Windows 起動時にログビューアを自動的に起動させることで、ログオン前からリアルタイム通知を受け取ることができます。 By configuring Log Viewer to start automatically at Windows startup, you can receive real-time notifications from the moment the PC boots.

  • 1 applippli-key のインストーラーを実行すると、ログビューアは Windows 起動時に自動起動するよう設定されます。 Running the applippli-key installer automatically registers Log Viewer to launch at Windows startup.
