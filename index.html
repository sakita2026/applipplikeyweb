Windows 多要素認証

Windowsへのログインを
より安全に、よりスマートに。

applippli-key は、スマートフォンを使った多要素認証で Windows へのアクセスを保護するセキュリティソリューションです。パスワードだけに頼らない、次世代の認証体験を提供します。

機能紹介

セキュリティと利便性を両立する、3つの機能

それぞれが独立して動作し、組み合わせることでより強固な認証環境を構築できます。

ワンタイムパスワード認証

スマートフォンの applippli Authenticator で生成した 6 桁の TOTP コードを使って Windows にログインします。30秒ごとに更新されるため、コードが盗まれても再利用できません。

TOTP / RFC 6238
ワンタッチログイン

スマートフォンをかざすだけでロック解除。PIN や TOTP コードの入力不要で、素早く安全にデスクトップへアクセスできます。

V2 新機能
ログビューア

認証ログをわかりやすく表示します。いつ、どのユーザーが、どの方法でログインしたかを一覧で確認でき、不正アクセスの早期検知を支援します。
